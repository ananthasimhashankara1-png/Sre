üîç System Monitoring & Performance

uptime  
Shows system uptime and load averages.  
Example: uptime

top / htop  
Real-time CPU, memory, and process monitoring.  
Example: htop

free -h  
Displays memory usage in human-readable format.  
Example: free -h

vmstat 1  
Virtual memory stats, updated every second.  
Example: vmstat 1

iostat -xz 1  
Disk I/O and CPU stats with extended info.  
Example: iostat -xz 1

dstat  
Combines CPU, disk, net, and memory stats.  
Example: dstat

sar -u 1 3  
CPU usage over time (3 samples, 1 sec apart).  
Example: sar -u 1 3

üßØ Troubleshooting & Debugging

ps aux  
Lists all running processes.  
Example: ps aux | grep nginx

strace  
Traces system calls of a process.  
Example: strace -p 1234

lsof  
Lists open files and ports.  
Example: lsof -i :80

netstat / ss  
Shows network connections and listening ports.  
Example: ss -tuln

ping / traceroute  
Tests network connectivity and path.  
Example: ping google.com

tail -f  
Follows live logs.  
Example: tail -f /var/log/syslog

journalctl -xe  
Shows recent systemd logs with errors.  
Example: journalctl -xe

‚öôÔ∏è Service & System Management

systemctl  
Manages systemd services.  
Example: systemctl restart nginx

hostnamectl  
Views or sets hostname.  
Example: hostnamectl set-hostname web1

df -h  
Shows disk usage in human-readable format.  
Example: df -h

du -sh  
Shows size of a directory.  
Example: du -sh /var/log

crontab -e  
Edits cron jobs for automation.  
Example: crontab -e

chmod / chown  
Changes file permissions and ownership.  
Example: chmod 755 script.sh

üß™ Text Processing & Scripting

grep  
Searches for patterns in files.  
Example: grep "error" /var/log/syslog

awk / sed  
Text processing and transformation.  
Example: awk '{print $1}' file.txt

cut  
Cuts fields from text.  
Example: cut -d':' -f1 /etc/passwd

find  
Finds files by name, size, or time.  
Example: find /var -name "*.log"

xargs  
Executes commands on input from another command.  
Example: find . -name "*.tmp" | xargs rm
